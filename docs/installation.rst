Installation
============

Requirements
------------

``mapca`` requires Python >= 3.5 and the following packages:

- nibabel >= 2.5.1
- nilearn
- numpy >= 1.15
- scikit-learn >= 0.22
- scipy >= 1.3.3

Install
-------

You can install ``mapca`` from PyPI::

    pip install mapca

Or install the latest development version from GitHub::

    pip install git+https://github.com/ME-ICA/mapca.git
